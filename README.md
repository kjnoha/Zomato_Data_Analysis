# Zomato Data Analysis

A data analysis project exploring restaurant data from Zomato, presented through an interactive Power BI dashboard.

## Project Contents

- `zomato_data.xlsx` — Raw dataset containing restaurant-level data (e.g., ratings, cuisines, cost, location, and other attributes typically found in Zomato listings).
- `Zomato_Insights_Dashboard.pbix` — Power BI report built on top of the dataset, providing visual insights into the data.

## What This Project Covers

This dashboard analyzes restaurant data to surface insights such as:
- Distribution of ratings across restaurants
- Cuisine popularity
- Cost-for-two trends
- Location-based restaurant patterns

## How to Use

1. Clone or download this repository.
2. Open `Zomato_Insights_Dashboard.pbix` in [Microsoft Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free download).
3. Use the dashboard's filters and visuals to explore restaurant trends.
4. `zomato_data.xlsx` contains the raw data if you want to extend or reanalyze it.

## Tools Used

- **Power BI** — for building the interactive dashboard
- **Excel (.xlsx)** — as the raw data source
